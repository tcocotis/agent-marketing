The user is the consultant referenced in the AI-Program-Evaluation-Scope.docx. They are responsible for all deliverables across both programs.

**Approach:** Work will be executed using AI agents — Claude-based or other commercially available agents. No custom code will be written. Deliverables will primarily be documents and workflows, possibly including .md files.

**How to apply:** When helping plan or execute deliverables, think in terms of AI agent workflows, prompt design, tool configuration, and process documentation — not software development. Keep all outputs practical and consultant-ready (proposals, requirements docs, vendor assessments, POC plans, reports).
